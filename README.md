Este repositório contém um sistema de recomendção de músicas baseado em conteúdo(TF-IDF), implementado em Python e divido em alguns arquivos:
- O notebook jupyter no qual está o passo a passo da construção do sistema.
- O arquivo em Python que utiliza o streamlit para criar um localhost e exibir a aplicação.
- Um arquivo de texto com os requerimentos para que a aplicação funcione, para roda-la em seu sistema basta clonar o repositório, e através do terminal cirar um ambiente virtual na pasta do projeto e rodar o seguinte comando: pip install -r requirements.txt
